# 청계 A01 · 종로 점심 맛집지도

전략기획파트 자율주행서비스 체험 일정에 맞춰 제작한 맛집 공유 페이지입니다.

## 배경

**4월 29일**, 전략기획파트 자율주행서비스 현장 체험의 일환으로 **청계 A01 자율주행버스** 탑승이 예정되어 있음.  
A01 노선(청계광장 ↔ 광장시장)을 따라 청계천 주변에 위치한 점심 맛집을 한눈에 볼 수 있도록 인터랙티브 지도 형태로 정리.
